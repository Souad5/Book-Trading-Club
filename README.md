# 📚 ShelfShare – Collaborative Book Sharing Platform (Team Project)

🔗 **Live Site:** [https://book-trading-club-delta.vercel.app]  

---

## 🚀 Overview

ShelfShare is a collaborative book-sharing platform that enables users to **lend, borrow, and discover books** within a community-driven ecosystem. The platform combines secure authentication, seamless payment integration, and intelligent assistance to enhance user experience and engagement.

---

## 🎯 Key Highlights (Why this project stands out)

- Developed a **secure authentication system** using NextAuth with protected routes  
- Integrated **Stripe payment workflow** for borrowing and lending transactions  
- Built scalable and reusable UI components with modern frontend architecture  
- Connected frontend with backend via **RESTful APIs**  
- Implemented an **AI-powered assistant** for book-related queries  
- Collaborated in a **team environment** using version control and structured workflows

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/your-username/shelfshare.git

# Navigate into the project
cd shelfshare

# Install dependencies
npm install

# Run development server
npm run dev
```
🔑 Environment Variables

Create a .env.local file:
NEXTAUTH_SECRET=your_secret
NEXTAUTH_URL=http://localhost:3000
STRIPE_SECRET_KEY=your_key
